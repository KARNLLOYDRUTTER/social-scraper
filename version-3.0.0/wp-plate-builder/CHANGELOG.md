# Changelog

## 1.7.0
- Built‑in SVG sizing (no theme CSS required).
- DVLA metrics applied (50×79mm chars, 11mm char spacing, 33mm group gap, 11mm margins).
- 4×4 plates render on two lines with correct line gap.
- Short 4/5/6 widths corrected; per-size maxlength enforced.
- HEX (Lambo) now true double‑ended hex, with proper clipping for fill/badges.
- Optional optical spacing for slim glyphs (1/I) to improve legibility.
- Large rear (533×152) size corrected.
